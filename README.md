> Эти 5 AI-сотрудников я обещал участникам вебинара **«Реальный AI для владельца бизнеса 2026»**. Каждый — готовый промпт-скилл, который превращает обычный AI (Claude, ChatGPT, Codex) в специалиста с конкретной ролью и алгоритмом работы.

**Они не пишут «креативные тексты вообще»** — они закрывают конкретные задачи бизнеса:

## Как этим пользоваться — 3 способа

### 🟢 Способ 1: Claude Code (рекомендуем — самый сильный)

Если у вас установлен **[Claude Code](https://claude.com/claude-code)** — самый удобный вариант. Скиллы устанавливаются один раз и доступны всегда по триггер-фразам.

#### Установка одной командой

**На Mac / Linux:**
```bash
cd ~/.claude/skills && \
  git clone https://github.com/{{GITHUB_USER}}/ai-employees-skills.git tmp-employees && \
  cp -r tmp-employees/skills/* . && \
  rm -rf tmp-employees && \
  echo "✅ 5 сотрудников установлены"
```

**На Windows (через Git Bash или WSL):**
```bash
cd "$HOME/.claude/skills" && \
  git clone https://github.com/{{GITHUB_USER}}/ai-employees-skills.git tmp-employees && \
  cp -r tmp-employees/skills/* . && \
  rm -rf tmp-employees && \
  echo "✅ 5 сотрудников установлены"
```

**Через PowerShell:**
```powershell
cd $env:USERPROFILE\.claude\skills
git clone https://github.com/{{GITHUB_USER}}/ai-employees-skills.git tmp-employees
Copy-Item tmp-employees\skills\* . -Recurse
Remove-Item tmp-employees -Recurse -Force
"5 сотрудников установлены"
```

#### Как вызывать после установки

Просто пишете в Claude Code — он сам подхватит нужного:
- «Проведи аудит инстаграма @kompaniya» → запустится **instagram-auditor**
- «Сделай КП для клиента из ниши X» → запустится **kp-generator**
- «Напиши сценарий рилса про Y» → запустится **reels-scriptwriter**
- «Обработай возражение клиента: цитата» → запустится **objection-handler**
- «Собери tone of voice по этим постам» → запустится **tone-of-voice-keeper**

Или явно: `/instagram-auditor`, `/kp-generator`, `/reels-scriptwriter`, `/objection-handler`, `/tone-of-voice-keeper`.

### 🟡 Способ 3: ChatGPT / Claude.ai в браузере (без CLI)

Если у вас нет Claude Code или Codex — работает в любом веб-чате (ChatGPT, Claude.ai, Gemini, локальный LLM).

#### Как использовать

1. Откройте файл нужного скилла на GitHub:
   - [instagram-auditor/SKILL.md](skills/instagram-auditor/SKILL.md)
   - [kp-generator/SKILL.md](skills/kp-generator/SKILL.md)
   - [reels-scriptwriter/SKILL.md](skills/reels-scriptwriter/SKILL.md)
   - [objection-handler/SKILL.md](skills/objection-handler/SKILL.md)
   - [tone-of-voice-keeper/SKILL.md](skills/tone-of-voice-keeper/SKILL.md)

2. Скопируйте всё содержимое (Raw → Ctrl+A → Ctrl+C)

3. Вставьте в чат **первым сообщением** с приставкой:

   ```
   Действуй строго по этой инструкции. Ниже — твоя роль и алгоритм работы:

   {вставленное содержимое SKILL.md}

   Начни с шага 0.
   ```

4. AI начнёт задавать вопросы по шагу 0. Отвечайте — он пройдёт по фреймворку и выдаст результат.

#### Pro-tip для ChatGPT

Сохраните каждый скилл как **Custom GPT** или **Project Instructions** — тогда не нужно копировать каждый раз.

В ChatGPT → My GPTs → Create:
- **Name:** «Аудитор Instagram»
- **Instructions:** вставить содержимое `instagram-auditor/SKILL.md`
- **Conversation starters:** «Проведи аудит профиля @...»

То же самое для остальных 4. После этого — выбираете нужного GPT и работаете.

## Что внутри каждого скилла

Каждый SKILL.md устроен по одной схеме:

1. **Frontmatter** — метаданные + триггер-фразы для авто-вызова
2. **Роль и принцип** — кто этот сотрудник и какова его главная цель
3. **Шаг 0: Сбор контекста** — какие вопросы задаёт пачкой перед работой
4. **Алгоритм** — пошаговый разбор задачи по фреймворку
5. **Формат финального ответа** — что именно выдаёт на выходе
6. **Чеклист качества** — проверка перед выдачей результата
7. **Anti-patterns** — что НЕ делать (защита от типичных ошибок AI)
8. **Связка с другими skills** — какой сотрудник логично подхватит работу следующим

Это не «промпт на 50 слов», а **полноценный системный промпт уровня платных GPT-агентов** — 200-400 строк методологии каждый.

## FAQ

**Сколько стоит этим пользоваться?**
Скиллы бесплатные. Платите только за подписку Claude / ChatGPT / Codex (если используете API). Для большинства задач хватит обычной платной подписки за $20/мес.

**Можно ли модифицировать под свою компанию?**
Да, нужно. После установки откройте `~/.claude/skills/<имя_скилла>/SKILL.md` и добавьте свой контекст: ниша, тон бренда, любимые формулировки. Чем точнее настроите — тем лучше работает.

**Что если результат плохой?**
Проверьте что ответили на все вопросы шага 0. 80% плохих результатов — из-за того что AI не знает о вашем бизнесе. Дайте ему: ЦА, цели, цифры, примеры — и он раскроется.

**А с открытым кодом / в проде нельзя использовать?**
Можно. Лицензия MIT — берёте, дорабатываете, используете как угодно.
